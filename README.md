
# Équipe Hopper - Robot suiveur de ligne

Projet de conception et de réalisation d'un robot autonome suiveur de ligne dans le cadre de l'UE Bureau d'Études, encadrée par M. Hughes Cassés.

## Matériel
- Arduino Uno
- Shield moteur DFRobot L293
- 2 moteurs DC
- 3 capteurs Pololu QTR
- Ball caster
- 2 Batteries Li-Ion

## Logiciel

- `courseVitesse` : version utilisé pour la course de vitesse.
- `courseRobustesse` : version optimiutilisésée pour la course de robustesse.
- `test` : programmes utilisés lors des phases de développement et de validation.
